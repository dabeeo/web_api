dabeeo web api

api를 사용하기 위한 간단한 설명입니다. 자세한 내용은 해당 파일을 참고하세요.

MapView : 지도그리기 (서버에서 지도파일 가져와서 지도 그리기)

  • 다비오 지도서버에서 지도를 다운로드하여 지도를 그리는 방법입니다.
  • indoorMapApi.MapView 클래스를 이용하여 지도를 그립니다.
  • 이 때 모든 지도 정보를 가져오므로 이 이후에는 반환된 값을 (MapDraw) 이용하여 지도를 제어할 수 있습니다.
  • 초기 option을 지정할 수 있습니다.

MapDraw : 지도 속성 및 지도 제어

  • MapView 호출시 mapDraw가 response로 반환하며 여기에 지도에 관한 정보들이 담겨있습니다. mapDraw의 속성와 메소드를 이용하여 지도 설정을 변경하거나 필요한 지도를 제어할 수 있습니다.

Event : api 호출시 반환되는 event

  • 지도제어 중에 층변경, 클릭, 모의주행완료 시 이벤트가 발생됩니다. 이벤트를 받아 이벤트내의 정보를 활용할 수 있습니다.
