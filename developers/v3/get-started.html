Getting Started


다비오맵스 API에서 지도를 불러오기 위해서는 해당 지도의 Client ID / Secret code를 알아야 합니다. 아래의 절차에 따라 정보를 확인하세요.


Issue Client ID / Secret

  1. 다비오맵스 실내지도플랫폼 > 실내지도 > 지도관리 메뉴로 이동합니다.
  2. 지도관리 목록에서 지도 정보를 클릭합니다.
  3. 상세 지도 정보에서 인증접근 정보를 확인합니다.
