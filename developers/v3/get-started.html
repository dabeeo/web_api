Getting Started


1. 다비오맵스 API를 불러옵니다.

1. Link dabeeo maps JS API.

* 다비오맵스 JS API의 공식 api 링크는 [다비오맵스 실내지도 플랫폼]에서 확인할 수 있습니다. * The official versoin of Dabeeomaps JS API is located at [Dabeeomaps platform].

2. 다비오맵스 플랫폼 서버에서 지도인증정보를 획득합니다.

2. Obtain map authentication information from Dabiomaps platform server.

* API에서 지도를 불러오기 위해서는 해당 지도의 Client ID / Secret code를 알아야 합니다. 아래의 절차에 따라 정보를 확인하세요. * To load a map from the API, you need to know the Client ID / Secret code of the map. Follow the steps below to check your information

Issue Client ID / Secret

  1. [다비오맵스 실내지도 플랫폼] > 실내지도 > 지도관리 메뉴로 이동합니다.
  2. [Dabeeomaps platform] goto > indoor map > indoor map management menu.
  3. 지도관리 목록에서 지도 정보를 클릭합니다.
  4. Click map information in the map management list.
  5. 상세 지도 정보에서 인증접근 정보를 확인합니다.
  6. Check the authentication access information in the detailed map information.
  7. 코드에 클라이언트아이디와 클라이언트시크릿을 입력합니다.
  8. Enter the client ID and client secret in the code.
      
        var authorization = new indoorMapApi.Authorization({
            clientId: "28AXw_veA2YbNKDP6poTpT",
            clientSecret: "70c540c169af62808f4da3709e988e06"
        });

3. 다비오맵스 JS API를 사용하여 지도를 렌더링합니다.

3. Render the map using Dabeeomaps JS API.

* 다비오맵스의 공식 api 링크는 [다비오맵스 실내지도 플랫폼] 에서 확인할 수 있습니다. * Dabeeomaps official JS API link is at [Dabeeomaps platform].
* API를 로딩하는 스크립트의 위치는 head, body등 어디에 위치해도 상관없으나 실행코드보다 항상 먼저 선언되어야 합니다.
* The location of the script that loads the API doesn't matter where it is located, such as head or body, but it must always be declared before the executable code.
실행 페이지와 전체 코드는 [examples index]를 참조하세요.
The example page and the full code are at [examples index].

4. 위 예제를 실행한 화면은 다음과 같습니다.

* 디폴트로 3d모드로 설정되어 있으며 마우스를 이용하여 이동, 확대, 회전등이 가능합니다.
