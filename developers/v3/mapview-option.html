MapView Option


MapView 생성시 초기 option을 지정합니다.


다비오 지도서버에 있는 지도를 불러와 그릴 때 초기 option을 지정할 수 있습니다.
zoom level 등 지도를 그리기 위한 설정이 여기에 해당합니다. 자세한 정보는 [API Reference의 Map Option]에 설명되어 있으니 이를 참조하세요.
옵션은 아래와 같이 설정할 수 있습니다. 

    
            let mapOptions = {
                authorization: authorization,
                canvasSize: { width: 800, height: 300 }, //size 설정
                isPoiSprite: true, //Poi를 항상 정면으로 보이게 함. 
                language: "en", //초기 poi 언어 설정 
                camera: "3d", //초기 카메라 설정
                zoom: 200, //초기 줌 설정
                center: {
                    x: 3030, // 지도 내 좌표 x 
                    y: 1505 // 지도 내 좌표 y
                },
                angle: {
                    vertical: 40,
                    horizontal: 0,
                    fixed: false
                }
            };
