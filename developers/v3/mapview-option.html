MapView Option


MapView 생성시 초기 option을 지정합니다.

Specifies initial options when creating MapView.


다비오 지도서버에 있는 지도를 불러와 그릴 때 초기 option을 지정할 수 있습니다.
You can specify the initial options when drawing a map from the Dabeeo map server.
초기 zoom값 등 지도를 그리기 위한 설정이 여기에 해당합니다.
This is the setting for drawing the map, such as the initial zoom value.
옵션은 아래와 같은 방법으로 설정할 수 있습니다. 전체 옵션에 대한 설명은 [API Reference의 Map Option]를 참조하세요.
Options can be set in the following ways. For a full description of the options, See [API Reference Map Option] 
    
        let mapOptions = {
            authorization: authorization,            // 인증 정보
            canvasSize: { width: 800, height: 300 }, // 캔버스 사이즈 설정
            isPoiSprite: true,                       // Poi를 항상 정면으로 보이게 함. 
            language: "en",                          // 초기 poi 언어 설정 
            camera: "3d",                            // 초기 카메라 모드 3d
            zoom: 200,                               // 초기 줌 설정
            center: {                                // 초기 중심 좌표 
                x: 3030,                             // 지도 내 좌표 x 
                y: 1505                              // 지도 내 좌표 y
            },
            angle: {                                 // 3d 모드인 경우의 각도 
                vertical: 40,                        // 수직 경사
                horizontal: 0,                       // 수평 경사
                fixed: false                         // 고정 여부
            }
        };
    

    
        let mapOptions = {
            authorization: authorization,            // authorization information
            canvasSize: { width: 800, height: 300 }, // canvas size
            isPoiSprite: true,                       // Poi face to front
            language: "en",                          // poi language
            camera: "3d",                            // poi camera mode 
            zoom: 200,                               // initial zoom value 
            center: {                                // center coordinate
                x: 3030,                             // x axis coordinate
                y: 1505                              // y axis coordinate
            },
            angle: {                                 // angle in 3d camera mode
                vertical: 40,                        // vertical tilt
                horizontal: 0,                       // horizontal tilt
                fixed: false                         // fixed or not
            }
        };
