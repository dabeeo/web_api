지도 불러오기 (서버에서 지도파일 가져와서 지도 그리기)

Loading a map


indoorMapApi.MapView를 이용하여 다비오 지도서버에 있는 지도를 불러와 지정된 영역에 지도를 그립니다.

Using indoorMapApi.MapView, draw a map in the designated area by loading the map in Dabeeo map server.


지도를 불러오기 위해서는 지도 사용을 위한 인증 정보가 필요합니다.
To load a map, you need authentication information to use the map.
MapView를 호출시 지도에 관한 모든 정보가 mapDraw 객체로 반환되므로 이를 이용하여 지도를 제어할 수 있습니다.
When MapView is called, all information about the map is returned as a mapDraw object, so you can use it to control the map.
MapView에 대한 자세한 정보는 [API Referece의 MapView] 에 설명되어 있으니 이를 참조하세요.
For more information about MapView, see [API Referece MapView].
