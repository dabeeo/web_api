Displaying HTML Elements in poi


MapDraw.setPOITag()를 이용하여 poi에 HTML Element를 표시할 수 있습니다.

You can display HTML Element in poi by using MapDraw.setPOITag().


이를 위해서 MapView 생성시 isUseTag를 true로 설정해야 하며, 지도영역위에 사용자 컴포넌트가 올라가는 경우 z-index를 설정해줘야 합니다.
For this, isUseTag must be set to true when creating MapView, and z-index must be set when a user component is placed on the map area.
인자로는 poi의 아이디와 HTML의 속성을 설정해주는 tagInfo가 있습니다. 옵션 설정에 대한 예제는 아래와 같습니다.
As arguments, there is the ID of poi and tagInfo that sets the attributes of HTML. An example of setting options is as follows.
자세한 설명은 [API Reference의 setPOITag]를 참조하세요.
For a detailed description, see [API Reference object click event]. 
    
        poiId: poi.id,
        tagInfo: {
            tag: dom1,
            width: 170,
            height: 70,
            pos: "BOTTOM",
            isResize: false
        }
