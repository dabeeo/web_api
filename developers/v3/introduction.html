다비오맵스 JS API 는 다비오맵스 실내지도 플랫폼 지도를 활용하여 지도 서비스를 쉽고 빠르게 구축 할 수 있게 JavaScript 형태로 공개한 기술입니다.


공식 api 링크는 다비오 사이트에 올라가 있습니다.
기능 개선 성능 향상이 실시간 업데이트되니, [다비오맵스 실내지도 플랫폼]에 연결된 공식 api 링크를 사용하시기 바랍니다.



Example Site


다비오 맵스 JS API를 쉽게 이해하고 활용을 돕고자 최신 버전의 다양한 JS 사용예제를 제공합니다.
주요 기능에 대하여 실행해 볼 수 있으며 실행되는 페이지의 전체 코드를 볼 수 있습니다. [Examples Site]를 참고하세요.

Dabeeo Maps Javascript API Classes


다비오맵스 JS API의 주요 클래스입니다. 자세한 내용은 [API Reference]를 참고하세요.

MapView : 지도그리기 (서버에서 지도파일 가져와서 지도 그리기)


다비오맵스 실내지도 플랫폼 서버에서 지도를 가져와서 지정된 영역에 지도를 그립니다.
indoorMapApi.MapView를 이용시 초기 옵션을 지정할 수 있습니다.
이 때 모든 지도 정보를 가져오므로 이 이후에는 반환된 값을 (MapDraw) 이용하여 지도를 제어할 수 있습니다.

MapDraw : 지도 속성 및 지도 설정 변경하기


MapView를 호출시 mapDraw가 반환되며 지도에 관한 정보들이 속성에 담겨있습니다.
이 정보를 바탕으로 메소드를 이용하여 지도를 제어하거나 필요한 지도정보를 사용자에게 보여줄 수 있습니다.

Event : api 호출시 반환되는 event


지도제어 중에 층변경, 클릭, 모의주행완료 시 이벤트가 발생됩니다. 이벤트를 받아 이벤트내의 정보를 활용할 수 있습니다.
