Dabeeo Maps Javascript API Classes


다비오맵스 JS API의 주요 클래스입니다. 자세한 내용은 [API Reference]를 참고하세요.
Below are the main classes of Dabiomaps JS API. See here for details [API Reference].

MapView : 지도그리기 (서버에서 지도파일 가져와서 지도 그리기)

MapView : Load a map (get a map from server and render a map)


다비오맵스 실내지도 플랫폼 서버에서 지도를 가져와서 지정된 영역에 지도를 그립니다.
It gets a map from Dabeeomaps indoor map platform server and draws a map in the designated area.
indoorMapApi.MapView를 이용시 초기 옵션을 지정할 수 있습니다.
Initial map options can be specified when using indoorMapApi.MapView.
이 때 모든 지도 정보를 가져오므로 이 이후에는 반환된 값을 (MapDraw) 이용하여 지도를 제어할 수 있습니다.
At this time, all map information is fetched, so after that, you can control the map using the returned value (MapDraw).

MapDraw : 지도 속성 및 지도 설정 변경하기

MapDraw : Change map properties and map settings


MapView를 호출시 mapDraw가 반환되며 지도에 관한 정보들이 속성에 담겨있습니다.
When MapView is called, mapDraw is returned and information about the map is contained in the property.
이 정보를 바탕으로 메소드를 이용하여 지도를 제어하거나 필요한 지도정보를 사용자에게 보여줄 수 있습니다.
Based on this information, you can use a method to control the map or show the necessary map information to the user.

Event : api 호출시 반환되는 event

Event : Event returned when api is called


지도제어 중에 층변경, 클릭, 모의주행완료 시 이벤트가 발생됩니다. 이벤트를 받아 이벤트내의 정보를 활용할 수 있습니다.
events occur when floor change, click, or simulation run is completed during map control. You can receive the event and utilize the information in the event.
