poi의 정보 표시하기


object-click 또는 poi-click 이벤트를 이용하여 지도의 POI내에 있는 정보를 사용자에게 표시할 수 있습니다.


다비오맵스 플랫폼에서 poi에 입력된 사용자 정보는 object가 클릭되거나 poi가 클릭될 때 event를 통해 전달됩니다.
이를 통해 poi의 여러가지 내용을 팝업창등을 통해 사용자에게 알릴 수 있습니다.
object 클릭시에는 object에 연결된 poi의 정보가 전달됩니다. 한개의 object에 여러개의 poi가 연결될 수 있습니다.
object 클릭 이벤트에 대한 자세한 설명은 [API Reference의 object click event]를 참조하세요.
poi 클릭 이벤트에 대한 자세한 설명은 [API Reference의 poi click event]를 참조하세요. 
    
        //네비게이션 옵션 설정
        mapDraw.setNavigationOption({
            lineColor: "#00008b",       // 길찾기 주행선 색상 
            lineZ: 40,                  // 길찾기 주행선 z축 높이 
            speedRate: 3,               // 모의주행 아이콘 속도
        });

경로를 그리는 방법은 아래 예제에서 보는 바와 같이 mapDraw.getRouteOn()을 호출하면 됩니다.
인자로는 출발지, 목적지, 층간이동수단, 경유지, 반환값에 대한 옵션을 지정할 수 있습니다.
출발지와 목적지는 반드시 지정하여야 하며 나머지는 선택사항입니다. 지정하지 않을 경우 디폴트값이 적용됩니다.경유지는 복수가 지원되므로 배열형태로 입력해야 합니다.
출발지와 목적지, 경유지는 poiID로 하거나 특정좌표 모두 가능하며, poi 속성을 그대로 넘겨도 가능합니다.
층간이동경로는 추천, 계단, 엘리베이터, 에스컬레이터 4가지 타입이 존재합니다. 지정하지 않는 경우 추천으로 선택됩니다. 추천은 최단경로입니다.
MapDraw.getRouteOn()에 대한 자세한 정보는 [API Reference의 MapDraw.getRouteOn]에 설명되어 있습니다. 
    
        //길찾기 경로 그리기
        result = mapDraw.getRouteOn(originPoi, destPoi, "recommendation", [], retResponse = true);

mapDraw.startRouteAnimation()으로 모의주행을 그릴 수 있습니다. 길찾기 경로를 따라 모의주행아이콘이 출발지에서 도착지까지 움직입니다.
모의주행아이콘에 대한 옵션은 앞서 얘기한 MapDraw.setNavigationOption()으로 줄 수 있으며
모의주행시 줌 설정과 카메라 설정, 모의주행중 층변경시 딜레이 설정과 도착지의 애니메이션 효과는 mapDraw.startRouteAnimation()에서 지정합니다.
MapDraw.startRouteAnimation()에 대한 자세한 정보는 [API Reference의 MapDraw.startRouteAnimation]을 참조하세요. 
    
        mapDraw.startRouteAnimation();
        mapDraw.stopRouteAnimation();
